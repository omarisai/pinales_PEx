Component OneWire
Component for the One Wire Protocol
Component Level: High
Category: SW-Communication
Methods:
(Methods are user-callable functions/subroutines intended for the component functions control. Please see the Embedded Components page for more information.)
  • Init - Initializes this device.

      ANSIC prototype: void %'ModuleName'%.Init(void) Init(void)

  • SendReset - Sends a reset to the bus

      ANSIC prototype: void SendReset(void)

  • Receive2 - Programs a read operation after the master send all in output buffer. Don't use a SendReset while the data is coming.

      ANSIC prototype: bool Receive2(uint8_t numOfBytes, uint8_t counter)
    • numOfBytes:uint8_t - number of bytes
    • counter:uint8_t - Number of bytes to receive from slave
    • Return value:bool - Returns FALSE if the program buffer is full or if count is greater than the free space of input buffer-

  • Count - Returns the number of elements stored on input buffer that are ready to read.

      ANSIC prototype: uint8_t Count(void)
    • Return Value:uint8_t - no hint

  • ProgramEvent - no hint

      ANSIC prototype: bool ProgramEvent(uint8_t key)
    • key:uint8_t - Key to identificate the event, applies only if OnProgramEvent is enabled. (Valid range 0 - 31)
    • Return value:bool - Returns false if the program buffer is full.

  • Waitms - Programs a pause between instruccions.

      ANSIC prototype: bool Waitms(byte key, uint8_t time_ms)
    • key:byte - Key to identify the source of the event.
    • time_ms:uint8_t - Value of time to wait.
    • Return value:bool - Returns false if the program buffer or the time buffer is full.

  • SendByte - Sends a single byte

      ANSIC prototype: bool SendByte(byte data)
    • data:byte - Variable to save the byte.
    • Return Value:bool - no hint

  • SendBytes - Sends multiple bytes

      ANSIC prototype: bool SendBytes(uint8_t *data, uint8_t count)
    • data: Pointer to uint8_t - Array of bytes to add to output stream.
    • count:uint8_t - Number of bytes to add to output stream. (Valid range 0 - 31)
    • Return Value:bool - no hint

  • GetBytes - Gets multiple bytes from the bus

      ANSIC prototype: bool GetBytes(uint8_t *data, uint8_t count)
    • data: Pointer to uint8_t - no hint
    • count:uint8_t - no hint
    • Return Value:bool - no hint

  • GetByte - Get a single byte from the bus

      ANSIC prototype: bool GetByte(uint8_t *data)
    • data: Pointer to uint8_t - no hint
    • Return Value:bool - no hint

  • GetError - Returns the error

      ANSIC prototype: void GetError(void)


Note: Some methods can be implemented as macros.
